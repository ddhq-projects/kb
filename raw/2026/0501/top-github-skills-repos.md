## Как начать

1. **Посмотреть что есть:** skills.sh leaderboard
2. **Найти нужное:** VoltAgent/awesome-agent-skills (1000+ indexed)
3. **Установить:** `npx skills add <owner/repo>`
4. **Создать свой:** skill-creator от Anthropic
5. **Поделиться:** push на GitHub → automatic discovery

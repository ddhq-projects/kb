# База знаний DDPA — raw source materials

Этот каталог содержит неизменяемые исходные документы (статьи, PDF, скриншоты), 
из которых AI-агенты синтезируют wiki-страницы в `src/content/docs/`.

## Структура

```
raw/
└── YYYY/       # Год (например, 2026)
    └── MMDD/   # Месяц + День (например, 0505 = 5 мая)
        └── file.{md,pdf}  # Исходные материалы
```

## Правила

- **READ ONLY** — после добавления файлы не изменяются
- Каждый файл должен содержать frontmatter с `source` (URL оригинала)
- Именование: описательное, транслитом или на английском, без пробелов

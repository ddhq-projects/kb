## Обзор

AI-ассистенты — это потребительские продукты с чат-интерфейсом на базе больших языковых моделей. В отличие от «голого» API, они предоставляют удобный UI, историю диалогов, загрузку файлов и дополнительные фичи.

## Сравнение основных ассистентов (2026)

### ChatGPT (OpenAI)
- **Модели:** GPT-5.2, GPT-4o, o4-mini
- **Интерфейс:** веб, мобильные приложения, десктоп
- **Ключевые фичи:** Code Interpreter, GPTs (кастомные боты), DALL·E, web search, Canvas, голосовой режим
- **Контекст:** 128K токенов
- **Плюсы:** лучшая экосистема, огромное сообщество, максимальная функциональность
- **Минусы:** цензура, блокировки в РФ, высокая цена Pro
- **Цена:** Free / Plus $20/мес / Pro $200/мес

### Claude (Anthropic)
- **Модели:** Claude 4 Opus / Sonnet
- **Интерфейс:** веб, iOS
- **Ключевые фичи:** Projects (рабочие пространства), Artifacts (интерактивные результаты), 200K контекст, анализ PDF/изображений
- **Контекст:** 200K токенов
- **Плюсы:** лучший для анализа документов и кода, качественный русский, безопасность
- **Минусы:** нет генерации изображений, нет веб-поиска, нет Android-приложения
- **Цена:** Free / Pro $20/мес / Max $100/мес

### Gemini (Google)
- **Модели:** Gemini 2.5 Pro / Flash
- **Интерфейс:** веб, Android, iOS
- **Ключевые фичи:** Deep Research, интеграция с Google Workspace (Gmail, Docs, Sheets), 1M+ контекст, YouTube анализ
- **Контекст:** 1M+ токенов
- **Плюсы:** самый большой контекст, интеграция с Google-сервисами, бесплатный Deep Research
- **Минусы:** нестабильное качество, галлюцинации
- **Цена:** Free / Advanced $19.99/мес (включён в Google One)

### Grok (xAI)
- **Модели:** Grok 3
- **Интерфейс:** веб (X.com), iOS
- **Ключевые фичи:** доступ к X/Twitter в реальном времени, минимальная цензура, генерация изображений (Aurora)
- **Плюсы:** реал-тайм данные, свобода ответов, юмор
- **Минусы:** меньше фич для продуктивности, меньше экосистема
- **Цена:** Free (ограниченный) / SuperGrok $30/мес

### Perplexity
- **Модели:** GPT-4o, Claude 4, Gemini 2.5, собственные
- **Интерфейс:** веб, мобильные приложения
- **Ключевые фичи:** поиск с цитированием источников, Collections, Spaces, Perplexity Pages
- **Плюсы:** лучший для research-задач, всегда с источниками, быстрый
- **Минусы:** не универсальный ассистент, слабее в творческих задачах
- **Цена:** Free / Pro $20/мес

### DeepSeek
- **Модели:** DeepSeek-V3, DeepSeek-R1 (Deep Think)
- **Интерфейс:** веб, iOS
- **Ключевые фичи:** Deep Think (reasoning), загрузка файлов, открытый исходный код моделей
- **Контекст:** 128K токенов
- **Плюсы:** отличный reasoning, очень низкая цена API, открытый код
- **Минусы:** меньшая экосистема, ограниченная мультимодальность, вопросы к приватности
- **Цена:** бесплатно

### GigaChat (Сбер)
- **Модели:** GigaChat 2.0
- **Интерфейс:** веб, API
- **Ключевые фичи:** российская экосистема, генерация изображений (Kandinsky), API для бизнеса, работа с документами
- **Плюсы:** доступен в РФ без VPN, оплата российскими картами, 19M пользователей, интеграция с продуктами Сбера
- **Минусы:** слабее ChatGPT/Claude по качеству, меньше международных интеграций
- **Цена:** Free (ограничен) / API: от 1.5 ₽/1K токенов

## Сценарии использования

| Задача | Рекомендация |
|--------|-------------|
| Универсальный ассистент | ChatGPT |
| Анализ документов и код | Claude |
| Исследования (research) | Perplexity, Gemini Deep Research |
| Работа с Google-документами | Gemini |
| Реал-тайм данные соцсетей | Grok |
| Reasoning / сложная логика | DeepSeek Deep Think, Claude |
| Бюджетное решение | DeepSeek (бесплатно), ChatGPT Free |

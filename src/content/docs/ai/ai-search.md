## Что такое AI-поиск

AI-поисковики — это инструменты поиска информации, которые вместо выдачи списка ссылок формируют развёрнутый ответ с источниками, объединяя данные из нескольких веб-страниц. Используют LLM для суммаризации и факт-чекинга.

---

## Основные AI-поисковики (2026)

### Perplexity
- **Тип:** AI-поисковик с цитированием
- **Модели:** собственные, GPT-4o, Claude 4, Gemini (на выбор)
- **Ключевые фичи:**
  - Мгновенный поиск с источниками
  - Collections — сохранение и организация находок
  - Spaces — совместное workspace для исследования
  - Pages — AI-генерация страниц из находок
  - Deep Research — многоходовой анализ темы
  - Pro Search — уточняющие вопросы перед поиском
- **Плюсы:** лучший для research, всегда с источниками, скорость
- **Минусы:** не всегда глубокий анализ, слабее в творческих задачах
- **Цена:** Free / Pro $20/мес

### ChatGPT Search
- **Тип:** поиск, встроенный в ChatGPT
- **Модели:** GPT-4o / GPT-5.2
- **Ключевые фичи:** встроен в ChatGPT-интерфейс, автоматический поиск при релевантном вопросе, цитирование
- **Плюсы:** часть экосистемы ChatGPT, плавный опыт
- **Минусы:** работает только внутри ChatGPT, менее специализирован
- **Цена:** включён в ChatGPT Plus ($20/мес)

### Google AI Overviews / Gemini Search
- **Тип:** AI-суммаризация в Google-поиске
- **Модели:** Gemini 2.5
- **Ключевые фичи:** автоматическая сводка наверху результатов, AI Mode (полный AI-поиск)
- **Плюсы:** встроен в Google — не нужно ничего устанавливать
- **Минусы:** нельзя контролировать, часто галлюцинации
- **Цена:** бесплатно

### You.com
- **Тип:** AI-поисковик с фокусом на приватность
- **Ключевые фичи:** Research, Genius, YouAgent, multi-model доступ
- **Плюсы:** приватность, выбор моделей, чиcтый интерфейс
- **Минусы:** меньше пользователей, хуже источники
- **Цена:** Free / Pro $20/мес

### Phind
- **Тип:** AI-поисковик для разработчиков
- **Ключевые фичи:** технический поиск, код в ответах, документация, StackOverflow-интеграция
- **Плюсы:** лучший для технических вопросов, поиск по коду
- **Минусы:** только технические темы
- **Цена:** Free / Pro $20/мес

### Glean — Enterprise Search
- **Тип:** AI-поиск по внутренним данным компании
- **Оценка:** $7.2B, ARR $200M
- **Ключевые фичи:** единый поиск по Google Docs, Slack, Jira, Confluence, Salesforce, GitHub, email; RAG-based; учитывает permissions
- **Плюсы:** лучший для enterprise knowledge management, самообучается на данных компании
- **Минусы:** только enterprise, не для личного пользования
- **Цена:** по запросу (enterprise)

### ЖИЖИ
- **Тип:** российский AI-поисковик с чатом
- **Ключевые фичи:** интеллектуальный поиск с ИИ-чатом, поиск по российским источникам
- **Плюсы:** работает в РФ без VPN, ориентирован на российский контент
- **Минусы:** молодой продукт, меньше возможностей чем Perplexity
- **Цена:** бесплатно (на старте)

## Как выбрать AI-поисковик

1. **Perplexity** — лучший выбор по умолчанию для большинства пользователей. Бесплатная версия уже хороша.
2. **ChatGPT Search** — если вы уже активно пользуетесь ChatGPT.
3. **Google AI Overviews** — для бытовых запросов, где не нужно углубляться.
4. **Phind** — для разработчиков, когда нужен код и техническая документация.
5. **You.com** — если важна приватность.

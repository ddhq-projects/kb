## AI-генерация изображений

### Midjourney
- **Тип:** генерация изображений по текстовому описанию
- **Интерфейс:** Discord, веб (Alpha)
- **Ключевые фичи:** художественный стиль, высокая детализация, V7 модель, референсы стиля, zoom out, pan
- **Плюсы:** лучшее качество для креативного дизайна, арт-направление
- **Минусы:** интерфейс через Discord, не для точных правок
- **Цена:** Basic $10/мес / Standard $30/мес / Pro $60/мес

### DALL·E / GPT Image (OpenAI)
- **Тип:** генерация и редактирование изображений
- **Интерфейс:** ChatGPT, API
- **Ключевые фичи:** GPT Image 1.5 — точное следование промпту, inpainting, 4K, контекстное редактирование
- **Плюсы:** встроен в ChatGPT, понимает естественный язык
- **Минусы:** менее художественный, чем Midjourney; цензура
- **Цена:** включён в ChatGPT Plus ($20/мес), API отдельно

### Adobe Firefly
- **Тип:** генерация для бизнеса и дизайна
- **Интерфейс:** веб, интеграция с Adobe Suite (Photoshop, Illustrator)
- **Ключевые фичи:** commercially safe (обучен на лицензионном контенте), Generative Fill, Generative Expand
- **Плюсы:** легально для коммерции, интеграция с дизайн-инструментами
- **Минусы:** менее креативный, чем Midjourney; нужна подписка Creative Cloud
- **Цена:** включён в Creative Cloud ($55+/мес)

### Flux 2 Pro
- **Тип:** фотореалистичная генерация
- **Интерфейс:** API, веб-сервисы (Replicate, fal.ai)
- **Ключевые фичи:** лучший фотореализм, точное следование промпту
- **Плюсы:** open-weight, можно запускать локально
- **Цена:** API: ~$0.05/изображение; локально — бесплатно (нужна GPU)

### Grok Aurora (xAI)
- **Тип:** генерация изображений через Grok
- **Плюсы:** минимальная цензура, генерация знаменитостей
- **Минусы:** нестабильное качество, только в Grok
- **Цена:** SuperGrok $30/мес

### Nano Banana (Google)
- **Тип:** AI-редактирование изображений
- **Интерфейс:** веб
- **Ключевые фичи:** «ИИ-Фотошоп» — замена объектов, изменение фона, 4K
- **Плюсы:** простой интерфейс, быстрые правки
- **Цена:** бесплатно (Google One)

## AI-генерация видео

### Sora 2 (OpenAI)
- **Тип:** text-to-video
- **Фичи:** до 8+ секунд, реалистичная физика, стилизация
- **Цена:** включён в ChatGPT Plus/Pro

### Veo 3.1 (Google)
- **Тип:** text-to-video
- **Фичи:** 4K, аудио-сопровождение, вертикальные форматы, длительность до 2 минут
- **Плюсы:** высокое качество, встроен в YouTube Shorts
- **Цена:** платно через Google AI Studio / Vertex AI

### Runway
- **Тип:** профессиональное AI-видео
- **Фичи:** Gen-4 модель, text-to-video, image-to-video, video-to-video, motion brush, Act-One (анимация лица)
- **Плюсы:** лучший инструмент для креативного видео, регулярные инновации
- **Цена:** Free / Standard $15/мес / Pro $95/мес

### Kling AI
- **Тип:** text-to-video
- **Фичи:** длинные видео, motion control, высокое качество движения
- **Плюсы:** один из лучших по качеству движения
- **Цена:** бесплатно (ограничения) / платные планы

### Synthesia
- **Тип:** AI-аватары для бизнес-видео
- **Фичи:** виртуальные ведущие, 140+ языков, шаблоны, замена лиц
- **Плюсы:** идеально для обучающих видео, demos, корпоративных коммуникаций
- **Цена:** Starter $22/мес / Creator $67/мес / Enterprise

## AI-генерация аудио и речи

### ElevenLabs
- **Тип:** синтез речи и голосовые клоны
- **Фичи:** 29 языков, Voice Cloning, Voice Design, Audiobook narration, Dubbing
- **Плюсы:** самое естественное качество речи, клонирование голоса
- **Цена:** Free / Starter $5/мес / Pro $22/мес / Scale $99/мес

### OpenAI TTS
- **Тип:** синтез речи через API
- **Фичи:** 6 голосов, HD качество, потоковая передача
- **Цена:** $15/1M символов (TTS-1), $30/1M символов (TTS-1 HD)

### Suno AI
- **Тип:** AI-генерация музыки
- **Фичи:** text-to-music, генерация песен (текст + музыка), стилизация
- **Цена:** Free / Pro $8/мес / Premier $24/мес

### Udio
- **Тип:** AI-генерация музыки
- **Фичи:** качественная музыка, custom lyrics, ремиксы
- **Плюсы:** лучшее качество музыки, чем Suno (субъективно)
- **Цена:** Free / Standard $10/мес / Pro $30/мес

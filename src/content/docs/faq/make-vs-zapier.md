## Быстрый ответ

- **Make** — лучший баланс возможностей и цены для сложной автоматизации
- **Zapier** — самый большой каталог интеграций, простота
- **n8n** — open-source, self-hosted, AI-узлы, гибкость для технарей

## Сравнение возможностей

| | Make | Zapier | n8n |
|--|------|--------|-----|
| Тип автоматизации | Визуальный сценарий | Триггер → Действие | Визуальный workflow |
| Ветвления | ✅ | ⚠️ (Paths) | ✅ |
| Циклы | ✅ | ⚠️ (Loops by Zapier) | ✅ |
| Обработка ошибок | ✅ (Error handlers) | ⚠️ (авто-повтор) | ✅ |
| Задержки | ✅ | ✅ | ✅ |
| Фильтры | ✅ | ✅ | ✅ |
| Код | ❌ | Python/JS (дорого) | ✅ (бесплатно) |
| AI | ❌ (только интеграции) | ❌ (только интеграции) | ✅ (AI-узлы) |

## Цены (для малого бизнеса)

| | Make | Zapier | n8n |
|--|------|--------|-----|
| Бесплатно | 1000 ops | 100 задач | ∞ (self-hosted) |
| Стартовый | $9/мес (10k ops) | $30/мес (750 задач) | $24/мес (2500 выполнений) |
| Бизнес | $18/мес | $103/мес | $72/мес |
| Enterprise | $36/мес | от $598/мес | кастом |

## Количество интеграций

- **Zapier:** 7000+
- **Make:** 2000+
- **n8n:** 400+

## Сценарии выбора

### Выбирай Zapier если:
- У тебя нет технического бэкграунда
- Нужна интеграция с редким сервисом (скорее всего есть в Zapier)
- Бюджет не критичен

### Выбирай Make если:
- Нужна сложная логика (ветвления, циклы, обработка ошибок)
- Хочешь видеть сценарий визуально
- Важен баланс цена/возможности

### Выбирай n8n если:
- Готов(а) к self-hosting
- Нужен AI в автоматизациях
- Команда техническая и нужна гибкость
- Бюджет ограничен

## Примеры сценариев

### Make: мониторинг упоминаний бренда
```
RSS-лента → Filter (ключевые слова) → AI-классификация → Google Sheets → Telegram-уведомление
```

### n8n: AI-дайджест новостей
```
RSS → HTML-парсинг → AI-суммаризация → Markdown-письмо → Email через Gmail
```

### Zapier: сбор заявок
```
Facebook Lead Ads → Google Sheets → Email-уведомление менеджеру
```

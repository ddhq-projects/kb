## Быстрый ответ

- **ChatGPT** — лучшая экосистема (GPTs, Code Interpreter, DALL-E, Search)
- **Claude** — лучший анализ, длинный контекст, безопасность, код
- **Gemini** — самый большой контекст, интеграция с Google, Deep Research

## По задачам

### Написание текстов
1. **Claude** — самый естественный стиль
2. **ChatGPT** — хорош, но иногда шаблонно
3. **Gemini** — неровный

### Анализ данных и документов
1. **Claude** — Projects + большой контекст
2. **Gemini** — 1M контекст, но хуже качество
3. **ChatGPT** — Code Interpreter для CSV/Excel

### Кодинг
1. **Claude** — стабильно высокое качество
2. **ChatGPT** — хорош, но уступает Claude
3. **Gemini** — хуже всех

### Креатив
1. **ChatGPT** — DALL-E + генерация
2. **Claude** — отличный текст, но нет изображений
3. **Gemini** — Imagen, но текст слабее

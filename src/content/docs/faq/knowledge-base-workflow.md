## Зачем вести базу знаний

- **Не помнить — а знать где найти** (external brain)
- **Onboarding** — новые люди получают контекст быстро
- **Асинхронная коммуникация** — меньше вопросов «а как сделать X?»
- **Накопление опыта** — не теряешь знания при уходе людей

## Инструменты

### Для личной базы знаний
| Инструмент | Подходит для |
|-----------|-------------|
| **Obsidian** | Zettelkasten, граф знаний, исследователи |
| **Notion** | Командная wiki, базы данных |
| **Anytype** | Приватный Notion-опыт |
| **Logseq** | Дневниковый подход, аутлайнер |

| Инструмент | Подходит для |
|-----------|-------------|

## Структура

### PARA (Tiago Forte)
- **Projects** — активные проекты
- **Areas** — зоны ответственности
- **Resources** — справочные материалы
- **Archive** — неактивное

### MOC (Map of Content)
- Создавай «карты содержания» — страницы со ссылками на связанные темы
- Не создавай глубокую иерархию папок

### Атомарные заметки
- Одна заметка = одна идея
- Связывай заметки перекрёстными ссылками
- Структура вырастает органически

## Рабочий процесс

### Захват (Capture)
1. Встретил полезную статью/идею → сразу в Inbox
2. Не думай о структуре на этом этапе
3. Используй веб-клиппер (Obsidian Web Clipper, Notion Web Clipper)

### Обработка (Curate)
1. Пересматривай Inbox регулярно (раз в неделю)
2. Перепиши своими словами (не копируй)
3. Разбей на атомарные заметки
4. Добавь связи с существующими заметками

### Выражение (Express)
1. Регулярно создавай MOC (карты содержания)
2. Синтезируй идеи из разных заметок
3. Пиши — статьи, решения, гипотезы

## Антипаттерны

- ❌ Коллекционирование без осмысления (архив, а не база знаний)
- ❌ Слишком глубокая иерархия папок (сложно найти)
- ❌ Копипаст без своей переработки (получается копия интернета)
- ❌ Перфекционизм в оформлении (важнее содержание и связи)

## Принципы

- **Progressive Summarization** — выделяй главное, делай резюме
- **Evergreen notes** — заметки должны улучшаться со временем
- **Связывай, а не классифицируй** — ссылки важнее папок
- **Пиши для будущего себя** — через год ты не вспомнишь контекст

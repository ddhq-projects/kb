## Быстрый ответ

- **Notion** — если нужна командная работа и базы данных
- **Obsidian** — если нужна личная база знаний с полным контролем над файлами
- **Anytype** — если нужен Notion-опыт, но с приватностью и оффлайн

## Подробное сравнение

### Данные и приватность

| | Notion | Obsidian | Anytype |
|--|--------|----------|---------|
| Где данные | Облако Notion | Локальная папка | Локально + P2P |
| Без интернета | ❌ Медленно | ✅ Полный доступ | ✅ Полный доступ |
| Шифрование | В покое (на сервере) | Файловая система | E2E |
| Экспорт | Markdown/CSV | Markdown (уже!) | JSON |

### Организация

| | Notion | Obsidian | Anytype |
|--|--------|----------|---------|
| Парадигма | Иерархия страниц + БД | Свободный граф + папки | Объекты + типы |
| Базы данных | ✅ Мощные | Dataview plugin | Наборы |
| Шаблоны | ✅ | ✅ | ✅ |
| Связи (ссылки) | Backlinks | Двунаправленные + граф | Двунаправленные + граф |

### Работа с медиа

| | Notion | Obsidian | Anytype |
|--|--------|----------|---------|
| Изображения | ✅ Встроено | ✅ (локально) | ✅ |
| PDF | ✅ Встроено | ✅ Встроено | ✅ |
| Embed (YouTube, Figma) | ✅ | через iframe plugin | Ограничено |

### Цена

| | Бесплатно | Платно |
|--|----------|--------|
| Notion | Да (с ограничениями) | Plus $10/мес |
| Obsidian | Да (полный функционал) | Sync $5/мес, Publish $10/мес |
| Anytype | Да (beta) | — |

## Древо решений

```
Тебе нужна коллаборация (команда)?
├── Да → Notion
└── Нет → Важна приватность (E2E)?
    ├── Да → Anytype
    └── Нет → Нужны базы данных?
        ├── Да → Notion (если облако) или Obsidian + Dataview
        └── Нет → Obsidian
```

## Можно ли использовать два сразу?

Да. Популярные связки:
- **Notion** для проектов и командной работы + **Obsidian** для личной базы знаний
- **Obsidian** для заметок и мышления + **Notion** для публикации и шаринга

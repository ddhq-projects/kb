## Что такое Госуслуги

Единый портал государственных услуг (ЕПГУ) — платформа для получения государственных и муниципальных услуг онлайн. Более 130 млн пользователей. Приложение «Госуслуги Авто», «Госуслуги Культура», «Госключ» (электронная подпись).

## Ключевые возможности

- **Подтверждённая учётная запись** — доступ ко всем услугам
- **Госключ** — бесплатная электронная подпись (ЭП) в приложении
- **Госуслуги Авто** — электронные права, СТС, штрафы, оплата
- **Робот Макс** — AI-помощник для навигации по услугам
- **Жизненные ситуации** — сценарии, объединяющие несколько услуг в одну

## Обзор

No-code платформы позволяют автоматизировать рутинные процессы, соединяя сервисы без написания кода.

## Make (бывший Integromat)

- **Модель:** визуальный сценарий (drag-and-drop с узлами и связями)
- **Интеграции:** 2000+
- **Плюсы:** визуальный редактор операций, обработка ошибок, дешевле Zapier, AI-модули, MCP-сервер
- **Минусы:** сложнее начать, чем Zapier
- **Цена:** бесплатно (1000 ops/мес) / Core $9/мес (10 000 ops)

**Сценарий:** При новом письме в Gmail → создать задачу в Todoist → отправить уведомление в Telegram.

## Zapier

- **Модель:** триггер → действие (линейные Zaps)
- **Интеграции:** 8000+
- **Плюсы:** самый большой каталог интеграций, простота настройки, AI-агенты
- **Минусы:** дорогой, ограниченная логика (без сложных ветвлений)
- **Цена:** бесплатно (100 задач/мес) / Starter $19.99/мес (750 задач)

**Сценарий:** Новая запись в Google Forms → добавить строку в Google Sheets → отправить email.

## n8n

- **Модель:** self-hosted или cloud, визуальные workflow
- **Интеграции:** 400+
- **Плюсы:** open-source, self-hosted бесплатно, AI-узлы (LangChain, LLM), код в узлах, $2.5B valuation
- **Минусы:** меньше готовых интеграций, сложнее хостинг
- **Цена:** бесплатно (self-hosted) / Cloud от $20/мес

**Сценарий:** RSS-ридер → AI-резюме статей через OpenAI → отправка в Telegram-канал.

## Web/App Builders

### Bubble
- **Тип:** полноценные веб-приложения без кода
- **Фичи:** drag-and-drop интерфейс, встроенная БД, workflows, плагины, AI-генерация, мобильные apps
- **Плюсы:** можно построить полноценный SaaS, огромное сообщество
- **Минусы:** крутая кривая обучения, vendor lock-in, не для простых сайтов
- **Цена:** бесплатно / Starter $25/мес / Growth $95/мес

### Webflow
- **Тип:** визуальный сайтостроитель для профессионалов
- **Фичи:** визуальный CMS, чистый HTML/CSS, SEO, 4.5M сайтов, $213M revenue
- **Плюсы:** профессиональный код на выходе, мощная CMS, анимации
- **Минусы:** сложный для новичков, цена
- **Цена:** от $14/мес (базовый сайт) / CMS $23/мес

### Framer
- **Тип:** интерактивные сайты с анимацией
- **Фичи:** AI-генерация из промпта (Wireframer), дизайн + motion, компоненты
- **Плюсы:** лучший для дизайнеров (Figma-подобный редактор), потрясающие анимации
- **Минусы:** слабее CMS, не для сложных веб-приложений
- **Цена:** Free / Mini $5/мес / Basic $15/мес / Pro $25/мес

## Enterprise

### Microsoft Power Automate
- **Тип:** RPA + облачная автоматизация
- **Фичи:** Copilot AI, интеграция с Microsoft 365, RPA (desktop flows), 1000+ коннекторов
- **Плюсы:** глубокая интеграция с экосистемой Microsoft, RPA-возможности
- **Цена:** от $15/user/мес

### Workato
- **Тип:** enterprise iPaaS
- **Фичи:** governance, reusable recipes, enterprise-grade security, 1000+ коннекторов
- **Плюсы:** лучший для крупных компаний, строгая governance-модель
- **Цена:** по запросу

## Рекомендации

- **n8n self-hosted** — для технических команд, нужна гибкость и AI
- **Make** — баланс цена/возможности, сложная логика
- **Zapier** — максимальное количество интеграций, простота
- **Bubble** — для создания MVP и веб-приложений без кода
- **Webflow** — профессиональные сайты с CMS
- **Framer** — дизайнерские лендинги с анимацией
- **Power Automate** — если компания на Microsoft 365

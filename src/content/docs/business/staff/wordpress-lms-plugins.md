## WordPress как LMS-платформа

WordPress с LMS-плагином — open-source и self-hosted альтернатива облачным LMS-платформам. Подходит компаниям, которым нужен полный контроль над данными, кастомизация и отсутствие регулярной подписки. Три лидера рынка — **LearnDash**, **LifterLMS** и **Tutor LMS** — закрывают большинство сценариев корпоративного обучения.

## Общие возможности всех трёх

- Drag-and-drop конструктор курсов
- Drip-контент (выдача уроков по расписанию)
- Сертификаты об окончании
- Пререквизиты (условия доступа к курсам)
- Интеграция с WooCommerce
- Отслеживание прогресса студента
- Монетизация курсов

## LearnDash

**Фокус:** enterprise-сегмент, корпоративное обучение, высшие учебные заведения.

**Преимущества:**
- 8 типов квизов с тонкой настройкой (банки вопросов, таймеры, случайный порядок)
- Масштабируется до 10 000+ студентов без деградации производительности
- Focus Mode — distraction-free интерфейс для учеников
- Мобильное приложение (доп. плата)

**Минусы:** нет бесплатной версии для тестирования; цена $199/год за 1 сайт.

## LifterLMS

**Фокус:** membership-сайты и комьюнити-обучение.

**Преимущества:**
- Бесплатный core с хорошим функционалом для старта
- Встроенные платежи (Stripe/PayPal) без WooCommerce
- Membership-логика: уровни доступа, подписки, триалы
- Хорош для создания платных программ и комьюнити

**Минусы:** ограниченная масштабируемость (~2 000 студентов); некоторые ключевые функции только в add-on.

## Tutor LMS

**Фокус:** быстрый старт и маркетплейсы курсов.

**Преимущества:**
- Самая щедрая бесплатная версия (почти всё, что нужно для старта)
- 10 типов квизов с современным UI
- Встроенный frontend-конструктор курсов (не нужно выходить в админку)
- Поддержка мультипреподавателей и разделения доходов — идеально для маркетплейса
- Лайфтайм-лицензия за $499 (без ежегодной оплаты)

**Минусы:** нет мобильного приложения; некоторые Pro-функции только в дорогих тарифах.

## Рекомендации

| Сценарий | Выбор | Почему |
|----------|-------|--------|
| Крупная компания, 500+ студентов, сложное тестирование | **LearnDash** | Масштабируемость, продвинутые квизы |
| Membership/подписка, старт бесплатно | **LifterLMS** | Бесплатный core, встроенные платежи |
| Быстрый старт, маркетплейс, бюджет | **Tutor LMS** | Щедрая бесплатная версия, лайфтайм |

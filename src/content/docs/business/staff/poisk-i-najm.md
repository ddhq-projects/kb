## Общая картина рынка

Рынок онлайн-рекрутинга в России практически полностью занят отечественными платформами. **hh.ru** — безусловный лидер с долей ~56% рынка и базой более 77 млн резюме. Основные конкуренты — SuperJob (≈30 млн резюме) и Rabota.ru (24 млн проверенных анкет). Для разных сегментов найма эффективны разные площадки.

## Топ-платформы для поиска сотрудников

### hh.ru (HeadHunter)

Универсальная платформа для квалифицированных специалистов всех уровней. Самая большая база резюме в России, продвинутые инструменты поиска, автоматизации воронки подбора и аналитики рынка труда. Подходит для компаний любого размера.

- База: 77+ млн резюме
- Доля рынка: ~56%
- Лучше для: квалифицированные специалисты, офисные позиции, middle/senior

### SuperJob

Фокус на массовом найме, включая рабочие специальности. Хороший охват по регионам, собственные зарплатные исследования и аналитика рынка.

- База: ≈30 млн резюме
- Лучше для: массовый найм, рабочие специальности, регионы

### Rabota.ru

Проверенные анкеты (24 млн), быстрые отклики. Хорошо работает в регионах, особенно для линейного персонала и начальных позиций.

- База: 24 млн проверенных анкет
- Лучше для: регионы, быстрые отклики, линейный персонал

## Нишевые и бесплатные платформы

### Avito Работа

Эффективен для начальных позиций и рабочих специальностей. Низкий порог входа — размещение вакансий от 200 ₽. Подходит для малого бизнеса и срочного найма.

### Работа России

Бесплатная государственная платформа с доступом к 10 млн резюме через Госуслуги. Дополнительные инструменты: электронный документооборот (ЭДО), интеграция с центрами занятости. Хороший вариант для компаний, работающих с госзаказом.

### Хабр Карьера

Лидирующая платформа для IT-найма с 700 тыс. качественных резюме разработчиков, тестировщиков, аналитиков. Аудитория — технические специалисты с подтверждёнными навыками. Инструменты: зарплатная аналитика, брендинг работодателя, тестовые задания.

## ATS и AI-платформы для рекрутинга

| Платформа | Ключевые возможности | Преимущества | Цена |
|-----------|---------------------|-------------|------|
| **Garmony AI** | AI-скрининг (точность 97%), агрегация всех источников | Снижение времени найма в 5–10 раз | Старт от 7 дней |
| **Huntflow** | Автоматизация воронки, CRM для рекрутеров | Полный цикл подбора | По запросу |
| **Potok** | Интеграция YandexGPT, HR-модули | Для среднего бизнеса | Запуск 1–2 недели |

### Garmony AI

AI-платформа, агрегирующая кандидатов из всех источников (hh.ru, SuperJob, соцсети, внутренняя база). Скрининг резюме с точностью 97%, автоматический подбор по требованиям вакансии. По заявлениям разработчиков, сокращает время закрытия вакансии в 5–10 раз.

### Huntflow

Профессиональная ATS (Applicant Tracking System) с CRM-логикой для рекрутеров. Полный цикл: от заявки на подбор до выхода сотрудника. Воронки, автодействия, интеграции с hh.ru и другими площадками, конструктор отчётов.

### Potok

ATS-платформа со встроенной YandexGPT для автоматизации рутинных задач. Подходит для среднего бизнеса: модули подбора, адаптации, кадрового учёта. Запуск за 1–2 недели.

## Рекомендации по стратегии найма

- **IT-специалисты**: Хабр Карьера + hh.ru + Garmony AI
- **Массовый найм**: SuperJob + Avito Работа + Работа России
- **Квалифицированные специалисты**: hh.ru + Huntflow/Potok
- **Бюджетный найм**: Работа России (бесплатно) + Avito (от 200 ₽)
- **Скоростной найм**: Garmony AI (агрегация + AI-скрининг)

Для большинства компаний оптимальна комбинация 2–3 платформ: одна универсальная (hh.ru) + одна нишевая под специфику + ATS для автоматизации.

## Notion
- **Тип:** всё-в-одном (заметки + БД + проекты + wiki)
- **Плюсы:** мощные базы данных, шаблоны, коллаборация, веб-клиппер, AI
- **Минусы:** не оффлайн (медленный без интернета), нет сквозного шифрования, vendor lock-in
- **Цена:** бесплатно / Plus $10/мес / Business $18/чел/мес
- **Лучше всего для:** командной wiki, проектов с базами данных, кто хочет «всё в одном»

## Obsidian
- **Тип:** локальный текстовый редактор с графом связей
- **Плюсы:** Markdown-файлы на диске (полный контроль), оффлайн, быстрый, 1000+ плагинов, граф знаний
- **Минусы:** нет коллаборации «из коробки», синхронизация платная, нет мобильного веб-клиента
- **Цена:** бесплатно / Sync $5/мес / Publish $10/мес
- **Лучше всего для:** личной базы знаний (Zettelkasten), исследователей, писателей, long-form заметок

## Другие инструменты

### Microsoft OneNote
- **Тип:** свободное оформление, бесплатный
- **Плюсы:** полностью бесплатно, бесконечный холст, интеграция с Microsoft 365, рукописные заметки
- **Минусы:** закрытый формат, слабая мобильная версия, нет Markdown
- **Лучше всего для:** студентов, пользователей Windows/Office

### Google Keep
- **Тип:** быстрые заметки с интеграцией Google
- **Плюсы:** максимальная простота, напоминания по времени и геолокации, интеграция с Google Workspace
- **Минусы:** только для заметок, не для long-form, нет папок (только метки)
- **Лучше всего для:** быстрых заметок, списков покупок, напоминалок

### Apple Notes
- **Тип:** встроенные заметки для экосистемы Apple
- **Плюсы:** бесплатно, iCloud-синхронизация, рукописные заметки (iPad), сканер документов, быстрый
- **Минусы:** только Apple, нет Markdown, нет версионирования
- **Лучше всего для:** пользователей iPhone/iPad/Mac, быстрых заметок

### Evernote
- **Тип:** ветеран рынка заметок
- **Плюсы:** отличное распознавание текста на изображениях, веб-клиппер, поиск по PDF
- **Минусы:** дорогой ($15/мес), устаревший интерфейс, потерял аудиторию Notion и Obsidian
- **Лучше всего для:** сканирования и поиска по документам

## Anytype
- **Тип:** локально-first, объектно-ориентированный
- **Плюсы:** локальное хранение (IPFS-like), E2E-шифрование, оффлайн, бесплатный, граф, наборы (типы объектов)
- **Минусы:** молодой продукт, меньше интеграций, нет API, сложнее кривая обучения
- **Цена:** бесплатно (beta)
- **Лучше всего для:** тех, кому нужен Notion-like опыт с приватностью Obsidian

## Logseq
- **Тип:** outliner с двунаправленными ссылками
- **Плюсы:** open-source, локальные Markdown/Org-mode файлы, дневник (journal-first), граф, PDF-аннотации
- **Минусы:** медленнее Obsidian, мобильное приложение слабее
- **Цена:** бесплатно
- **Лучше всего для:** дневникового подхода, аутлайнера, Roam-альтернативы

## Рекомендации к выбору

| Сценарий | Инструмент |
|----------|-----------|
| Командная wiki, БД, проекты | **Notion** |

| Приватность + объектная модель | **Anytype** |
| Дневник + аутлайнер + open-source | **Logseq** |

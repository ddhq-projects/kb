## GTD (Getting Things Done) — David Allen

**Суть:** Освободи мозг от хранения задач — запиши всё во внешнюю систему.

### 5 шагов GTD
1. **Сбор (Capture)** — записывай всё, что приходит в голову, в Inbox
2. **Обработка (Clarify)** — для каждой задачи реши: сделать, делегировать, отложить или удалить
3. **Организация (Organize)** — разложи по проектам, контекстам, срокам
4. **Обзор (Reflect)** — еженедельный review всех списков
5. **Действие (Engage)** — выполняй задачи по контексту и энергии

**Инструменты:** Todoist, Things, Nirvana, OmniFocus

## PARA — Tiago Forte (Building a Second Brain)

**Суть:** Организуй цифровую жизнь по 4 категориям: Projects, Areas, Resources, Archive.

- **Projects (Проекты)** — активные задачи с конкретным сроком и результатом
- **Areas (Сферы)** — зоны ответственности без конечной даты
- **Resources (Ресурсы)** — темы интереса, справочные материалы
- **Archive (Архив)** — всё неактивное

**Принцип:** Не создавай пустых папок заранее — только под то, что нужно сейчас.

**Инструменты:** Notion, Obsidian, любая файловая система

## Zettelkasten

**Суть:** Атомарные заметки, связанные ссылками — эмерджентная структура знаний.

- **Атомарность** — одна заметка = одна идея
- **Связи** — каждая заметка ссылается на другие
- **Свои слова** — пересказывай, не копируй
- **Структура растёт** — не создавай иерархию заранее

**Инструменты:** Obsidian, Logseq, Roam Research, Zettlr

## Second Brain (CODE)

**Метод** от Tiago Forte:
- **C**apture — сохраняй интересное
- **O**rganize — раскладывай по PARA
- **D**istill — выделяй суть (прогрессивное резюмирование)
- **E**xpress — создавай и делись

## Как выбрать

| Стиль работы | Методология |
|-------------|------------|
| Много задач, нужен контроль | **GTD** |
| Цифровой хаос в файлах/заметках | **PARA** |
| Исследования, писательство, учёба | **Zettelkasten** |
| Творчество + продуктивность | **Second Brain** |

## Комбинирование

Можно комбинировать:
- **GTD** для задач + **PARA** для файлов/заметок
- **PARA** для организации + **Zettelkasten** внутри Resources

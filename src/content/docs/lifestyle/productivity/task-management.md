## Ключевые критерии выбора

При выборе таск-менеджера учитывай:
- **Объём задач** — личный список или командная работа 50+ человек
- **Платформы** — доступность на всех устройствах (iOS, Android, Web, Desktop)
- **Методология** — совместимость с GTD, Kanban, Agile
- **Интеграции** — календарь, email, Slack, API

## Основные сервисы

### Todoist
- **Для кого:** личная продуктивность, малые команды
- **Плюсы:** быстрый ввод на естественном языке, карма-система, шаблоны проектов
- **Минусы:** ограниченные возможности для Kanban на бесплатном тарифе
- **Цена:** бесплатно / Pro $5/мес / Business $8/мес

### TickTick
- **Для кого:** альтернатива Todoist с календарём и Pomodoro в одном окне
- **Плюсы:** встроенный Pomodoro-таймер, календарь, Eisenhower Matrix, трекер привычек, командные списки
- **Минусы:** хуже интеграции, чем у Todoist
- **Цена:** бесплатно / Premium ~$3/мес

### Taskfire
- **Для кого:** AI-ориентированная продуктивность
- **Плюсы:** встроенный ИИ-помощник, изучение паттернов работы, умные рекомендации по приоритетам
- **Минусы:** молодой продукт, меньше интеграций
- **Цена:** бесплатно (early access)

### Things 3
- **Для кого:** пользователи Apple, личная продуктивность
- **Плюсы:** лучший UI/UX, отличная GTD-совместимость, одноразовая покупка
- **Минусы:** только Apple, нет коллаборации, нет веб-версии
- **Цена:** Mac $50 / iPhone + Watch $10 / iPad $20

### Linear
- **Для кого:** продуктовые команды, стартапы
- **Плюсы:** скорость работы, клавиатурная навигация, Cycles, Roadmaps
- **Минусы:** только для команд, избыточен для личного использования
- **Цена:** бесплатно до 10 чел / Business $8/чел/мес

### ClickUp
- **Для кого:** универсальный инструмент для команд
- **Плюсы:** всё-в-одном (документы, чат, цели, тайм-трекинг)
- **Минусы:** сложный интерфейс, медленный
- **Цена:** бесплатно / Unlimited $7/мес / Business $12/мес

### Asana
- **Для кого:** средние и крупные компании
- **Плюсы:** Workflow Builder, Portfolios, Goals, отличная визуализация
- **Минусы:** дорогой, избыточен для малых команд
- **Цена:** бесплатно до 10 чел / Premium $13/чел/мес

### Motion — AI-планирование
- **Для кого:** руководители и специалисты с высокой загрузкой встреч
- **Плюсы:** динамическое расписание на базе AI — автоматически перестраивает календарь, расставляет приоритеты задач, учитывает дедлайны
- **Минусы:** дорогой, только календарно-ориентированный подход, нет Kanban
- **Цена:** от $19/мес (Individual) / Business от $15/чел/мес

### ClickUp AI
- **Встроен в ClickUp**
- **AI-фичи:** генерация задач, авто-саммари, AI-поиск по проектам, шаблоны
- **Плюсы:** глубоко интегрирован в экосистему ClickUp, экономит время на рутине
- **Минусы:** доступен только на платных тарифах, всё ещё уступает специализированным AI-инструментам
- **Цена:** +$7/чел/мес к тарифу ClickUp

## Как выбрать

| Сценарий | Рекомендация |
|----------|-------------|
| Личная продуктивность, все платформы | **Todoist** |
| Личная продуктивность, только Apple | **Things 3** |
| Личная продуктивность + Pomodoro + календарь | **TickTick** |
| AI-планирование календаря и встреч | **Motion** |
| Стартап, продуктовая команда | **Linear** |
| Команда, нужно «всё-в-одном» | **ClickUp** |
| Enterprise, управление портфелем проектов | **Asana** |
